**Edit B — LoadAddOn replacement (inside OpenApplyDialog, lines 148-151):**

The current code inside OpenApplyDialog (starting at line 147) is:
```lua
  local function OpenApplyDialog()
    if not LFGListApplicationDialog or not LFGListApplicationDialog_Show then
      pcall(LoadAddOn, "Blizzard_LFGList")
      pcall(LoadAddOn, "Blizzard_LookingForGroupUI")
    end
```

Replace lines 149-150 with a backwards-compatible local + two pcall lines:
```lua
  local function OpenApplyDialog()
    if not LFGListApplicationDialog or not LFGListApplicationDialog_Show then
      local _LoadAddOn = (C_AddOns and C_AddOns.LoadAddOn) or LoadAddOn
      pcall(_LoadAddOn, "Blizzard_LFGList")
      pcall(_LoadAddOn, "Blizzard_LookingForGroupUI")
    end
```

The local `_LoadAddOn` is declared inside the `if` block where it is used, keeping scope minimal. On Midnight 12.x, `C_AddOns.LoadAddOn` is used. On hypothetical older clients, falls back to the global `LoadAddOn`. The `pcall` wrapper is preserved for defensive error handling.
